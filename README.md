
# Inventory App

A lightweight inventory management system built with Next.js and Firebase.

## Key Features
- Real-time inventory tracking

## Getting Started
1. Clone repo:
 `git clone https://github.com/username/inventory-app.git`
 
2. Install deps: 
`npm install`

3. Setup env vars (`.env` file)

4. Run:
 `npm start`

## Tech Stack
- Frontend: Next.js
- Backend: Node.js
- Database: Firebase
- UI: React + MUI
